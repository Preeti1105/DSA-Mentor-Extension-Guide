# DSA Mentor – Chrome Extension

A Chrome extension designed to support users while solving DSA problems directly on coding platforms.

The extension provides structured hints, complexity guidance, personal notes, bookmarking, and lightweight progress tracking without leaving the coding environment.

---

## Features

- Structured problem hints

- Time & Space complexity guidance

- Personal notes per problem

- Bookmark / revision support

- Progress dashboard

---

## Tech Stack

- JavaScript
- HTML
- CSS
- DOM Manipulation
- Chrome Extension Architecture
- Browser Local Storage

---

## How It Works

1. Open coding platform
2. Extension reads problem information
3. Injects custom panel
4. Displays hints and complexity
5. Stores user progress locally
